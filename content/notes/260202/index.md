# 真的假的，只有我這樣？

要說真的以下發生的事情認爲只有自己這樣似乎也不太算，畢竟我有個唸心理系的姐姐，時時刻刻提醒我「我有夠怪」，所以要說自己不知道，那也不至於。

# 我的超級分享慾

## 在家庭中

或許是家庭因素吧，我自認算是很有分享慾的人，我總是想跟人分享最近的近況或是學到的新知巴拉巴拉。

爲什麼說是家庭因素呢？ 不同於普通亞州家庭，我和我的家人算是有很多 Family time。即使已經長大成人了，與家庭的連結還是不少。我和家姐仍然時刻與家長們（父親與母親）回報自身的近況。也不能算是例行公事或是被要求的，從小到大我們都會很自然的分享，所以對於 "分享" 近況這件事在我們的生活是很日常的。

我想這樣的分享也造就我們與父母親良好且健康的關係，我認爲就是有這樣的分享，我們與普通亞州家庭有著相當不同的家庭氛圍。雖然我無法完全贊同父母的教育方式，但無形中建立的  Family time 或許是一件不容易的事情，能與父母親經常性的分享近況，這應該是很難得的！

## 除了在家庭中

我是一個熱愛學習各種新知的人，但學完各種新知偶爾只是學了似乎特別沒意思，所以我會將我學到的新知分享給我覺得會用得上的人。

至於所謂的新知是什麼呢？ 其實也就是一些大可不必知道的事情啦xddd 例如我前幾個禮拜在看 r7 2700x 怎麼在我的 Proxmox VE 上開 VM 時能最大化單核性能時發現 2700x 的 8 Core 竟然是 4+4 Core 然後共用 L3 Cache。知到的當下我立刻分享給我的大學同學。（這不是很有趣嗎！！！）


# 無時無刻在0元腦中創業

最讓家姐詬病的可能就是每次去哪，我都與我父親無時無刻計算這家店的成本或利潤。Ok, I know 這可能聽起來真的很無聊，吃個飯還計算人家店的成本真的很好笑，但我覺得瞭解一家店有沒有在賺錢對我來說真的太有趣了。

這個 0 元創業偶爾還會失敗，例如我曾經篤定認爲蝦皮店到店一定倒光光，沒想到我輸慘了哈哈哈。

不過我覺得就是這樣不經意的去瞭解別人的生意，正是建立自己的洞察能力，更能去觀察市場的趨勢！我從小就有創業夢想，我覺得開民宿或膠囊旅館又或是室內設計公司也是我一直以來的夢想，雖然這花費的資金並不是我目前能負擔的，但我仍然努力朝這個方向邁進，現在我能做的就是仔細的觀察市場的動向，時時刻刻觀察這樣的公司爲什麼能賺錢，又或是爲什麼失敗。
