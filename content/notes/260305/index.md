# 維度一：從家庭角度

雖然每次在外面看到家長教育搗蛋的小孩，我看了不僅是直搖頭還覺得恐怖。但 Jabee 我其實很喜歡小孩的（近期特別迷雙胞胎 [@mumuboza]()），我覺得我堂外甥不哭的時候也很可愛哈哈哈哈。

總體來說，我還是希望未來我能組一個家庭，有兩個小孩，最好一男一女。我甚至在很久以前就幫我女兒取好英文名了-Amber，源自 <<俠盜聯盟>> 這部港片中的 Amber，由張靜初飾演（沒辦法，這部港片張靜初演得太帥了xdd）。

其實現在有不少所謂的頂客家族，但其實據我觀察下來，家庭中少了孩子還是會少下滿多快樂的，雖然養小孩真的還滿花錢的，但快樂不是錢買得到的！

當然以上還是建立在有經濟的基礎上，Jabee 我認為沒錢還生小孩的父母滿不負責任的。當然有人就會問啦，如果未來我的老婆不想生小孩呢？我認為領養小孩也是一個很不錯的選擇，自己生的未必比較好，重點還是在有沒有能力扶養小孩。

# 維度二：從生活環境的角度

認識 Jabee 我的就知道，我太愛日本了，我希望未來可以去日本生活。雖然台灣這個地方是一個我熟悉有朋友的地方，但不管是交通、氣候、生活舒適度，我認為都大輸日本。

從交通來看，台灣真的太多摩托車了，我實在受不了一堆摩托車的地方，雖然我有一台重機，但我仍然認為台灣的交通很大一部分被摩托車拖垮，亂鑽、亂衝、不排隊都是問題，另外台灣對於人行道的建設真的是少得可憐，日本是連荒郊野外都會設置人行道，真的是行人天堂。

從氣候來說，我比較喜歡冷冷的天氣，也許北海道會相當適合我，在夏天還能有涼爽的氣氛誰不愛呢？

從生活舒適度來看的話，日本超市的生食或蔬菜價錢都是遠低於台灣的，房價當然也是遠低於台灣，我可不想未來在台灣偏鄉買個小房子還得花上超過千萬的預算。

希望未來我能在日本有一間自己的小別墅，也許不用靠近市區，也許他不像豪宅一樣大，但只要不是像台北的"好窄"，那我覺得我就非常滿意了。另外想搬去日本的原因也是因為我喜歡車子，目前日本進口車的關稅是 0，在買進口車上面能降低我的成本，我未來也想買一些日本車，以上的因素搬到日本去再合適不過了。

# 維度三：工作

雖然現在讀 CS，未來從中正畢業後大概也在系統廠或是 IC 廠工作，但不論是台商或外商的工作其實都不算我的終極目標，因為我從小最想做的工作應該是建築師或室內設計師。

我其實到現在偶爾還是會看一些室內設計的影片，特別是一些極小宅或是豪宅設計。不過我們總說興趣不能當飯吃，我覺得認清現實還是比較重要。我確實對於 CS 是有一些天賦的，對於美感可能也沒什麼能贏過別人的，所以高中開始我就很確定我要讀 CS。

但在未來，我希望我能開一間室內設計公司，完成自己的夢想。
